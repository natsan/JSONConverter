JSONConverter
Пакеты:
1. yandexconverter
	классы:
	1.1 FileHandler: содержит функции чтения и записи из файла. Считывает данные
	в виде списка строк. Записывает данные также в виде списка строк.
	1.2. JSONConverter: -преобразует список строк в json-объект (url, content, title);
			    -преобразует список json-объектов (url, content, title) в 
			    в список json-объектов (url, snippet). Попутно удаляются
			    html-символы, title объединяется с content, результат
			    урезается до 300 символов (до точки в предложении).
			    -преобразует список json-объектов (url, (длsnippet) к списку строк.
			    (для конвертации используется внешняя библиотека Jackson JSON (http://jackson.codehaus.org/)
	1.3. JSONTest: Здесь производится запуск примера. На входе файл "intern-task.json.filtered",
	в котором хранится список json-объектов (url, content, title), далее этот список
	преобразуется в список json-объектов (url, snippet), который записывается в файл 
	intern-task_modified.json".
2. jsonobject
	классы:
	2.1. InputJSONObject: представляет собой класс, содержащий три поля:
			      url, content и title. Представляет собой json-объект.
			      Данные json-объекты необходимо считать из файла и
			      преобразовать в json-объект класса OutputJSONObject и
			      записать в файл
	2.2. OutputJSONObject: представляет собой класс, содержащий два поля:
			       url, snippet (snippet - это объединённый title
			       и content, урезанные до 300 символов). Список данных
			       json-объектов записывается в файл.
